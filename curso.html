HTML

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), a linguagem de código padrão utilizada para estruturar e exibir o conteúdo de páginas web. Ela funciona como um "esqueleto" para os sites, utilizando etiquetas (tags) para organizar textos, imagens, links e outros elementos, sendo interpretada pelos navegadores para apresentar o resultado visualmente ao usuário. O HTML funciona como a estrutura de uma página da web, utilizando um conjunto de etiquetas (tags em inglês e atributos) para organizar o conteúdo, como textos, imagens e links. Um navegador de internet lê o código HTML, interpreta as etiquetas e as exibe como uma página web visual para o utilizador. Cada etiqueta tem um propósito específico, como o parágrafo (p) que define o texto como um parágrafo, ou a imagem (img), que insere uma imagem na página.

Para quem esta começando parece dificil, parece complicado, mas essa é a parte mais facil, é basicamente memorisar a posição e o formato da pagina, o html em si, é como a forma da pagina, com ela voçê tem a base.

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem usada para definir o estilo visual de um documento web, como páginas HTML, permitindo controlar cores, fontes, espaçamentos e alinhamentos. O CSS separa o conteúdo (HTML) da sua apresentação, o que torna o código mais organizado e eficiente, sendo uma das linguagens fundamentais da internet.

Agora esse é mais com,plicado, o css é bem extenso muita informação, e aqui, qualquer coisa que voçê errar a pagina corrompe, e é muito facil errar, mas é questão de memorisar onde tem que ficar cada coisa, eu demorei um pouco para pegar o jeito nesse, mas ainda tenho bastante para aprender.

JAVASCRIPT

JavaScript (JS) é uma linguagem de programação dinâmica utilizada principalmente para tornar páginas da web interativas e dinâmicas, adicionando conteúdo que se atualiza, animações, formulários e outros elementos interativos. Também pode ser usada para o lado do servidor (backend) com ambientes como o Node.js. O JavaScript é uma das tecnologias essenciais da web, trabalhando em conjunto com o HTML (estrutura) e o CSS (estilo) para criar a experiência do usuário na internet.

Nessa parte é muito diferente JavaScript é super complicado, super dificil, e bem problematico, porem é dinamico e a tecnologia deixou muito mais facil de usar essa linguagem de programação, alias vale ressaltar que html e css não são linguagens de programação e sim linguagens de formatação, eu aprendi batante nesse meu curso e vou mostrar um pouco na proxima pagina.
